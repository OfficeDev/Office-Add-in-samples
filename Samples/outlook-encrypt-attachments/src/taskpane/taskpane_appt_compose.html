Demo Instructions

This add-in will showcase how to use event-based activation to handle the OnAppointmentAttendeesChanged, OnAppointmentTimeChanged and OnAppointmentAttachmentsChanged event handlers.

For more information, see Configure your Outlook add-in for event-based activation

  • Add any required or optional attendee. A notification message will be added that shows a running tally of how many required and optional attendees have been invited to the meeting request
  • Add any Distribution List and see an additional notification message that warns you there are DLs that are receiving this meeting request.
  • Make a change to the appointment date and/or time. A notification message will appear displaying the original date and time. This is very handy when editing existing appointments so you can easily refer back to the original date!
  • Add any attachment
  • A copy of the attachment will be added with "encrypted_" prefixed to the original file name
  • The encrypted attachment will then be decrypted and added as a third attachment with "decrypted_" prefixed to the original file name
