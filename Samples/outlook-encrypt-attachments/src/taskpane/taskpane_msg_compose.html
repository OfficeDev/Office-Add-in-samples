Demo Instructions

This add-in will showcase how to use event-based activation to handle the OnMessageComposeHandler and OnMessageAttachmentsChanged event handlers.

For more information, see Configure your Outlook add-in for event-based activation

  • Add any attachment
  • A copy of the attachment will be added with "encrypted_" prefixed to the original file name
  • The encrypted attachment will then be decrypted and added as a third attachment with "decrypted_" prefixed to the original file name
