Create and explore data types

Use this tool to create data types or to explore data types that already exist in a workbook.

To create a data type, use the Data type dropdown list to choose a data type (string, double, boolean, entity, web image, or formatted number), enter data in the displayed fields, and then select the Set data button. The data type will be inserted into the currently active cell on the worksheet.

To explore existing data types, select a cell on the worksheet that contains data, and then select the Get data button. The data type information will be displayed in the task pane fields.
