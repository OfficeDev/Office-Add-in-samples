Office Add-ins sample: Verify the sensitivity label of a message

Please sideload your add-in to see app body.

Test the sample today!

  1. Create a new message.
  2. Add an attachment or the email address of a fictitious legal team employee at Fabrikam (for example, eli-legal@fabrikam.com).
  3. Select Send.
  4. A dialog appears to notify you that the sensitivity label has been set to Highly Confidential and legalhold@fabrikam.com has been added to the Bcc field. Once you've verified this, select Send Anyway.

To learn more about the sensitivity label API, see Manage the sensitivity label of your message or appointment in compose mode.
