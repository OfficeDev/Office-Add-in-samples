Get, Set, and Edit OOXML

Select sample content in the document or add content and then select it. Then click 'Get...' below to display the Office Open XML (OOXML) markup for the selection in the text area of this task pane (using getSelectedDataAsync).

When you click 'Insert...' below, the Set function in this app coerces the OOXML shown in the text area to add it to the document at the active selection point.



Use the text area as a scratch pad: You can use the task pane text area to test your edited markup. Copy and paste in that text area as needed. Then, click the 'Insert...' button to test your markup.

Tip: Display formatting marks (such as paragraph marks) when selecting document content to ensure that you select only what's needed for your markup. On the Home tab, in the Paragraph group, click the paragraph mark icon.
